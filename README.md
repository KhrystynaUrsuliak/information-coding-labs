# Лабораторна робота №6

## Передача інформації та виявлення помилок

## Мета роботи

Навчитися:

* моделювати передачу інформації через канал з помилками;
* досліджувати вплив шуму на передавання даних;
* реалізовувати прості методи виявлення та виправлення помилок;
* оцінювати ймовірність помилки;
* аналізувати ефективність захисного кодування.

---

## Теоретичні відомості

При передачі інформації через канал зв’язку можливі **помилки**.

Канал може змінювати передані біти, наприклад:

```text
0 → 1
1 → 0
```

---

## Модель каналу

Найпростіша модель — **двійковий симетричний канал (BSC)**.

Ймовірність помилки:

$$
p = P(\text{bit flip})
$$

тобто кожен біт змінюється з імовірністю $p$.

---

## Ймовірність помилки

Ймовірність помилки передачі залежить від:

* довжини повідомлення;
* ймовірності помилки каналу;
* наявності кодування.

---

## Виявлення помилок

Прості методи:

* біт парності
* контрольна сума

---

## Виправлення помилок

Найпростіший код:

**повторювальний код (repetition code)**

```text
0 → 000
1 → 111
```

Декодування:

* за правилом більшості

---

## Технічне завдання

Необхідно:

* змоделювати передачу повідомлення через канал з помилками;
* реалізувати канал з імовірністю помилки $p$;
* дослідити вплив помилок без кодування;
* реалізувати простий код (парність або repetition code);
* порівняти результати передачі з кодуванням і без нього.

## КРОК 1. Підготовка повідомлення

Створити двійкове повідомлення.

Наприклад:

```python
import random

def generate_message(n=1000):
    return [random.choice([0, 1]) for _ in range(n)]
```

---

## КРОК 2. Моделювання каналу

Реалізувати канал з помилками:

```python
def transmit(message, p=0.1):
    import random
    received = []
    for bit in message:
        if random.random() < p:
            received.append(1 - bit)
        else:
            received.append(bit)
    return received
```

---

## КРОК 3. Оцінка помилки без кодування

Порівняти:

* передане повідомлення
* отримане повідомлення

Обчислити кількість помилок.

---

## КРОК 4. Реалізація коду з повторенням

```python
def encode_repetition(message):
    encoded = []
    for bit in message:
        encoded.extend([bit, bit, bit])
    return encoded
```

---

## КРОК 5. Декодування

```python
def decode_repetition(message):
    decoded = []
    for i in range(0, len(message), 3):
        chunk = message[i:i+3]
        decoded.append(1 if sum(chunk) >= 2 else 0)
    return decoded
```
