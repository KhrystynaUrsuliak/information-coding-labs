# Лабораторна робота №3

## Оптимальне кодування (алгоритм Хаффмана)

## Мета роботи

Навчитися:

* будувати оптимальний префіксний код для джерела інформації;
* реалізовувати алгоритм Хаффмана;
* обчислювати середню довжину коду;
* порівнювати ефективність кодування з ентропією джерела;
* аналізувати наближення до межі Шеннона.

---

## Теоретичні відомості

При передачі інформації необхідно представити символи джерела у вигляді кодових слів.

Нехай джерело генерує символи з імовірностями $p_i$.
Кожному символу ставиться у відповідність кодове слово довжини $l_i$.

Середня довжина коду визначається як

$$
L = \sum_{i=1}^{n} p_i l_i
$$

---

## Теорема Шеннона

Для оптимального коду виконується:

$$
H(X) \le L < H(X) + 1
$$

де

* $H(X)$ — ентропія джерела
* $L$ — середня довжина коду

---

## Префіксний код

Код називається **префіксним**, якщо жодне кодове слово не є префіксом іншого.

Це дозволяє однозначно декодувати повідомлення.

---

## Алгоритм Хаффмана

Алгоритм Хаффмана будує оптимальний префіксний код для заданого розподілу ймовірностей.

Основна ідея:

1. Вибрати два символи з найменшими ймовірностями
2. Об’єднати їх у новий вузол
3. Повторювати до побудови дерева

У результаті отримується бінарне дерево, яке задає код.

---

## Технічне завдання

Необхідно:

* отримати частоти символів (з ЛР1 або згенерованого джерела з ЛР2);
* побудувати код Хаффмана;
* визначити кодове слово для кожного символу;
* обчислити середню довжину коду $L$;
* порівняти $L$ з ентропією $H$.

## КРОК 1. Отримання частот символів

Обчислити частоти появи символів у тексті.

Ймовірність символу:

$$
p_i = \frac{n_i}{N}
$$

---

## КРОК 2. Побудова дерева Хаффмана

Реалізувати алгоритм Хаффмана.

Приклад структури вузла:

```python
class Node:
    def __init__(self, symbol=None, prob=0, left=None, right=None):
        self.symbol = symbol
        self.prob = prob
        self.left = left
        self.right = right
```
| Поле   | Значення                  |
| ------ | ------------------------- |
| symbol | символ (A, B, ...)        |
| prob   | ймовірність цього символу |
| left   | ліва гілка                |
| right  | права гілка               |

---

## КРОК 3. Генерація кодів

Обійти дерево та призначити коди:

* ліве ребро — 0
* праве ребро — 1

Приклад:

```python
def generate_codes(node, prefix="", codebook={}):
    if node.symbol is not None:
        codebook[node.symbol] = prefix
    else:
        generate_codes(node.left, prefix + "0", codebook)
        generate_codes(node.right, prefix + "1", codebook)
    return codebook
```

---

## КРОК 4. Обчислення середньої довжини коду

Обчислити:

$$
L = \sum p_i l_i
$$

де $l_i$ — довжина коду символу.

---

## КРОК 5. Обчислення ентропії

Використати функцію з ЛР1:

$$
H = -\sum p_i \log_2 p_i
$$

---

## КРОК 6. Порівняння результатів

Порівняти:

* ентропію $H$
* середню довжину коду $L$

Обчислити різницю:

$$
L - H
$$

---

## КРОК 7. Додатковий експеримент

Порівняти:

* природний текст
* випадковий текст

Побудувати код Хаффмана для обох випадків.

Порівняти:

* довжини кодів
* ефективність кодування
