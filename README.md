# Лабораторна робота №1

## Ентропія джерела інформації

## Мета роботи

Навчитися:

* обчислювати ентропію Шеннона для текстових джерел інформації;
* аналізувати статистичні властивості природних мов;
* досліджувати залежності між символами у тексті;
* оцінювати надлишковість джерела інформації;
* порівнювати ентропію природних і випадкових текстів.


## Теоретичні відомості

Ентропія Шеннона — це міра невизначеності джерела інформації.

Для джерела з імовірностями появи символів $p_i$ ентропія визначається формулою

$$
H(X) = - \sum_{i=1}^{n} p_i \log_2 p_i
$$

де

* $p_i$ — ймовірність появи символу
* $n$ — кількість символів алфавіту

Ентропія вимірюється у **бітах на символ**.


## Ентропія нульового порядку

Якщо враховуються лише частоти символів, обчислюється **емпірична ентропія нульового порядку**

$$
H_0 = -\sum p_i \log_2 p_i
$$

Ця модель не враховує залежності між символами.


## Ентропія першого порядку

Якщо враховуються залежності між сусідніми символами, обчислюється **ентропія першого порядку**.

Для цього використовуються **біграми** (пари символів).

Ентропія біграм може бути оцінена за формулою

$$
H_1 = - \sum p_{ij} \log_2 p_{ij}
$$

де

* $p_{ij}$ — ймовірність появи біграми.

У природних мовах зазвичай

$$
H_1 < H_0
$$

оскільки між символами існують статистичні залежності.


## Максимальна ентропія

Максимальна ентропія для алфавіту з $N$ символів:

$$
H_{max} = \log_2 N
$$


## Надлишковість джерела

Надлишковість визначається як

$$
R = 1 - \frac{H}{H_{max}}
$$

Вона показує, яка частина інформації є **надлишковою**.


## Технічне завдання

Необхідно виконати експериментальне дослідження ентропії текстових джерел.


## Вимоги до текстових даних

Для отримання коректних статистичних результатів необхідно використовувати **достатньо великі тексти**.

Мінімальний обсяг тексту:

```
не менше 50 000 символів
```

Бажано використовувати тексти обсягом **100 000 символів або більше**.

## Покроковий план виконання


## КРОК 1. Підготовка тексту

Необхідно:

* перевести текст у нижній регістр
* видалити пунктуацію
* видалити пробіли (якщо аналіз проводиться лише для букв)

Аналіз проводити **на рівні символів**.


## КРОК 2. Обчислення ентропії символів

Обчислити частоти появи символів.

Ймовірність символу:

$$
p_i = \frac{n_i}{N}
$$

де

* $n_i$ — кількість появ символу
* $N$ — загальна кількість символів

Після цього обчислити

$$
H_0
$$


## КРОК 3. Обчислення ентропії біграм

Сформувати послідовність **біграм**.

Приклад:

```
text = hello

біграми:

he
el
ll
lo
```

Обчислити ентропію біграм $H_1$.


## КРОК 4. Аналіз різних мов

Для кожного тексту обчислити:

* $H_0$
* $H_1$

Мінімум для трьох мов:

* українська
* англійська
* китайська (або інша мова з іншим алфавітом)

---

## КРОК 5. Перемішування тексту

Перемішати символи тексту випадковим чином.

Після цього знову обчислити ентропію.

Порівняти:

* ентропію початкового тексту
* ентропію перемішаного тексту



## КРОК 6. Дослідження збіжності ентропії

Обчислити ентропію для різних довжин тексту.

Наприклад:

```
100 символів
500 символів
1000 символів
5000 символів
10000 символів
```

Побудувати графік залежності ентропії від довжини тексту.


## КРОК 7. Випадковий текст

Згенерувати **випадковий текст** тієї ж довжини та того ж алфавіту.

Наприклад, для англійського тексту:

```
abcdefghijklmnopqrstuvwxyz
```

Згенерувати випадкову послідовність символів цього алфавіту.

Після цього:

1. обчислити $H_0$
2. обчислити $H_1$

Порівняти результати з природним текстом.



## Очікуваний результат

Зазвичай спостерігається:

| Тип тексту       | H₀                      | H₁           |
| ---------------- | ----------------------- | ------------ |
| природний текст  | менша                   | значно менша |
| випадковий текст | близька до максимальної | ≈ H₀         |
