## Теоретичні відомості

Джерело інформації — це модель, яка породжує послідовність символів із заданого алфавіту.

У найпростішому випадку символи генеруються **незалежно**, імовірність появи кожного символу не залежить від попередніх. Таке джерело називають **джерелом без пам’яті**.

Якщо ймовірність появи наступного символу залежить від попереднього символу, джерело має **пам’ять 1-го порядку**. Така модель описується **ланцюгом Маркова першого порядку**.



## Матриця ймовірностей переходів

Для джерела з пам’яттю 1-го порядку задається **матриця переходів**

$$
P = {p_{ij}}
$$

де

* $p_{ij}$ — ймовірність переходу від символу $i$ до символу $j$

Для кожного рядка матриці має виконуватися умова нормування:

$$
\sum_j p_{ij} = 1
$$



## Стаціонарне джерело

Джерело називається **стаціонарним**, якщо його статистичні характеристики не змінюються з часом.

Для стаціонарного марковського джерела матриця переходів є сталою під час генерації всієї послідовності.


## Джерело без пам’яті та джерело з пам’яттю

Для джерела **без пам’яті** ймовірність символу визначається лише загальним розподілом:

$$
P(X_k = x_i) = p_i
$$

Для джерела **з пам’яттю 1-го порядку**:

$$
P(X_k = x_j \mid X_{k-1} = x_i) = p_{ij}
$$

Тобто наступний символ залежить від попереднього.



## Ентропія джерела

Для аналізу згенерованої послідовності можна використовувати ентропію символів:

$$
H(X) = - \sum_{i=1}^{n} p_i \log_2 p_i
$$

де

* $p_i$ — ймовірність появи символу
* $n$ — кількість символів алфавіту

Ентропія вимірюється у **бітах на символ**.

## КРОК 3. Генерація послідовності символів

Використати матрицю переходів для генерації послідовності символів.

Приклад:

```python
import random

def generate_sequence(matrix, length=1000, alphabet=['A', 'B', 'C', 'D']):
    sequence = [random.choice(alphabet)]

    for _ in range(length - 1):
        prev_index = alphabet.index(sequence[-1])
        next_symbol = random.choices(alphabet, weights=matrix[prev_index])[0]
        sequence.append(next_symbol)

    return ''.join(sequence)
```

Згенерувати послідовність довжиною не менше 1000 символів.


## КРОК 4. Обчислення ентропії згенерованої послідовності

Для отриманої послідовності обчислити ентропію символів.

Приклад:

```python
from collections import Counter
import math

def calculate_entropy(text):
    frequencies = Counter(text)
    total = len(text)
    entropy = -sum((count / total) * math.log2(count / total) for count in frequencies.values())
    return entropy
```

Обчислити значення:

$$
H_0
$$

для згенерованої послідовності.
